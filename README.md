# 📚 Library Management System

A web-based **Library Management System** developed using **Python (Flask)** and **SQLite**. This application provides an efficient way to manage library books and user accounts through a simple and user-friendly interface.

---

## 📖 Project Overview

The Library Management System is designed to simplify library operations by allowing users to register, log in, and manage book records. The project demonstrates backend development using Flask, database integration with SQLite, and responsive web design using HTML and CSS.

---

## ✨ Features

- 👤 User Registration and Login
- 🔒 Forgot Password functionality
- 📚 Add and Manage Library Books
- 🔍 Search Books
- 📋 View Available Books
- 💾 SQLite Database Integration
- 🎨 Clean and Responsive User Interface

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Missyben/Library-Management-System.git
```

### Navigate to the Project Folder

```bash
cd Library-Management-System
```

### Install Flask

```bash
pip install flask
```

### Run the Application

```bash
python app.py
```

### Open Your Browser

```
http://127.0.0.1:5000
```
